<h2>Getting Started</h2>

<h3>Prerequisites</h3>
<ol>
  <li>Windows 7 / Windows 8 / Windows 10 / Windows 11</li>
  <li>Visual Studio 2016 up to 2026</li>
  <li>.NET Framework</li>
</ol>

<hr>

<h2>Installing</h2>

<details>
  <summary><b>Clone / Download Zip</b></summary>
  <br>
  Use the green "Code" button at the top of the repository to clone or download the project as a ZIP file.
</details>

<details>
  <summary><b>Click open .sln file</b></summary>
  <br>
  Navigate to the project folder and double-click the <code>.sln</code> file to open it in Visual Studio.
</details>

<details>
  <summary><b>Go to Server Explorer</b></summary>
  <br>
  In Visual Studio, go to <b>View > Server Explorer</b> to manage your database connections.
</details>

<details>
  <summary><b>Right Click SignUp.mdf and go to Properties</b></summary>
  <br>
  Locate your database file in the explorer, right-click it, and select properties to view its details.
</details>

<details>
  <summary><b>Copy connection String</b></summary>
  <br>
  In the Properties window, highlight and copy the full <b>Connection String</b> text.
</details>

<details>
  <summary><b>Open DatabaseConnection.cs on RailwayEnquiryRepository</b></summary>
  <br>
  Locate this file within the repository structure in your Solution Explorer.
</details>
